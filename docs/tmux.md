# tmux Configuration Reference

Source: [`tmux.conf`](../tmux.conf), [`tmux-theme.conf`](../tmux-theme.conf)

Symlinked to `~/.tmux.conf` and `~/.tmux-theme.conf`. Sources `~/.server-configs-generated/tmux.compat.conf` for version-adaptive settings.

---

## Prefix

**`Ctrl+a`** (not the default `Ctrl+b`).

To send `Ctrl+a` to a nested tmux or application: `Ctrl+a Ctrl+a`.

---

## General Settings

| Setting | Value | Purpose |
|---------|-------|---------|
| `history-limit` | `50000` | Scrollback buffer lines |
| `base-index` | `1` | Windows start at 1, not 0 |
| `pane-base-index` | `1` | Panes start at 1, not 0 |
| `renumber-windows` | `on` | Close window 2 → window 3 becomes 2 |
| `mouse` | `on` | Click panes, resize, scroll |
| `escape-time` | `10` | Milliseconds to wait after Escape (fast for vim) |
| `allow-rename` | `off` | Don't let programs rename windows |
| `mode-keys` | `vi` | Vi-style copy mode |

## Keybindings

### Splits & Windows (require prefix)

| Key | Action |
|-----|--------|
| `prefix \|` | Vertical split (opens in current path) |
| `prefix -` | Horizontal split (opens in current path) |
| `prefix c` | New window (opens in current path) |

Default split bindings (`"` and `%`) are unbound.

### Pane Navigation

| Key | Prefix Required | Action |
|-----|----------------|--------|
| `prefix h` | Yes | Select pane left |
| `prefix j` | Yes | Select pane down |
| `prefix k` | Yes | Select pane up |
| `prefix l` | Yes | Select pane right |
| `Alt+Left` | No | Select pane left |
| `Alt+Right` | No | Select pane right |
| `Alt+Up` | No | Select pane up |
| `Alt+Down` | No | Select pane down |

### Window Switching (no prefix)

| Key | Action |
|-----|--------|
| `Shift+Left` | Previous window |
| `Shift+Right` | Next window |

### Pane Resizing (require prefix, repeatable)

| Key | Action |
|-----|--------|
| `prefix H` | Resize left by 5 |
| `prefix J` | Resize down by 5 |
| `prefix K` | Resize up by 5 |
| `prefix L` | Resize right by 5 |

The `-r` flag makes these repeatable — press prefix once, then tap the letter multiple times.

### Session

| Key | Action |
|-----|--------|
| `prefix S` | Session/window tree picker (zoomed) |

### Config

| Key | Action |
|-----|--------|
| `prefix r` | Reload tmux config (displays confirmation message) |

## Copy Mode

Enter copy mode with `prefix [`. Uses vi keybindings.

| Key | Action |
|-----|--------|
| `v` | Begin selection |
| `y` | Copy selection and exit copy mode |

The `y` key copies to the system clipboard using a cross-platform chain:

```
pbcopy → wl-copy → xclip → xsel
```

First available tool wins. On headless servers with no clipboard, the copy stays in the tmux buffer.

## Status Bar

```
┌─────────────────────────────────────────────────────────────┐
│ SESSION  1:bash  2:vim*                    main  14:30      │
└─────────────────────────────────────────────────────────────┘
```

| Position | Content | Style |
|----------|---------|-------|
| Left | Session name `#S` | Bold, dark text on cyan (`colour39`) bg |
| Center | Window list `#I:#W` | Current: bold cyan bg; others: gray |
| Right | Git branch + time `%H:%M` | Gray text |

- Left max width: 20 characters
- Right max width: 60 characters
- Dark background: `colour235`; foreground: `colour252`
- Light background: `colour254`; foreground: `colour238`
- Theme selection uses `detect-theme`; tmux also reloads the palette on attach and on `client-light-theme` / `client-dark-theme` hooks in tmux 3.6+.

## Compatibility (`tmux.compat.conf`)

Generated by `setup.sh` based on detected tmux version. May include:

| Setting | Condition | Purpose |
|---------|-----------|---------|
| `default-terminal` | Always | Terminal type (`tmux-256color` or `screen-256color`) |
| `terminal-overrides` | tmux ≥ 2.2 | True color support |
| `allow-passthrough` | tmux ≥ 3.3 | Pass escape sequences to terminal |
| `set-clipboard` | tmux ≥ 2.6 | OSC 52 clipboard integration |
